# NOVA VPN 🦅

Самый простой и понятный VPN для твоего телефона. Помогает открывать любые заблокированные сайты и приложения в один клик. Работает быстро, зарядку не жрёт.

---

## 📥 КАК СКАЧАТЬ И УСТАНОВИТЬ

Установка займет ровно 1 минуту. Просто делай по шагам:

### Шаг 1. Скачиваем файл
Нажми на синюю ссылку ниже, прокрути открывшуюся страницу чуть-чуть вниз и в блоке **Assets** нажми на файл, который заканчивается на **`.apk`**:

👉 **[СКАЧАТЬ NOVA VPN (ЖМИ СЮДА)](https://github.com/SatoshiNova/NOVA-VPN/releases/tag/app)** 👈

---

### Шаг 2. Устанавливаем на телефон
1. Как только файл скачается, нажмите кнопку **«Открыть»** (или найдите его в шторке уведомлений / папке «Загрузки»).
2. Если телефон ругается и пишет: *«В целях безопасности установка заблокирована»* — **не пугаемся**, это нормально для любых программ не из Google Play.
3. Нажмите кнопку **«Настройки»** прямо в этом всплывающем окне и включите тумблер **«Разрешить установку из этого источника»** (или «Разрешить для этого браузера/проводника»).
4. Вернитесь назад и нажмите **«Установить»**.

---

### Шаг 3. Запуск
1. Найдите иконку **NOVA VPN** на экране телефона.
2. Откройте приложение и нажмите одну большую кнопку **«Подключиться»**.
3. Нажмите **«ОК»**, когда телефон спросит разрешение на создание VPN-подключения.

Все, можно пользоваться! 🎉
